Algorithmic Adventures: Virtual Bistro Simulation

A comprehensive simulation of a restaurant system, showcasing object-oriented programming, memory management, and advanced data structures.

The Virtual Bistro project is a comprehensive simulation of a restaurant system where I have progressively built and enhanced various components, moving from basic menu item representation to a dynamic, real-time kitchen management system. Over the course of seven projects, I have developed my skills in object-oriented programming, memory management, and advanced data structures, culminating in the creation of a sophisticated kitchen simulation.

Project 1 begins with the foundational Dish class, implementing basic object-oriented principles like encapsulation, constructors, and formatted output for menu items. This serves as the groundwork for building a functional restaurant system. In Project 2, I expand the simulation to include dynamic customer interactions and real-time order processing, incorporating inheritance, event-driven programming, and possibly data management for persistent state.

Project 3 advances this by introducing dietary accommodations, such as vegetarian or gluten-free options, and refines order processing through enhanced kitchen workflow management. Here, I leverage inheritance and polymorphism, ensuring that the system can handle various dish categories dynamically and adjust for dietary needs. Project 4 refines this further by using abstract base classes and pure virtual functions, making the system more scalable and allowing for polymorphic dish management.

By Project 5, I incorporate kitchen station management, introducing classes like KitchenStation and StationManager to simulate real kitchen operations. This project emphasizes memory management and dynamic data handling using linked lists and vectors, demonstrating advanced C++ techniques for efficient resource management. Project 6 optimizes the dish preparation process through queues, ensuring that orders are processed in sequence while accommodating dietary restrictions and ingredient availability.

Finally, Project 7 integrates a Binary Search Tree to manage recipes based on difficulty levels, pushing my understanding of data structures further. This project also tests my ability to handle dynamic memory and tree operations, demonstrating the use of advanced algorithms and operator overloading.

Together, these seven projects illustrate my growing expertise in C++, from fundamental OOP principles to complex data structures and real-time simulation management, culminating in a flexible and efficient restaurant simulation system. Through each iteration, I've enhanced my problem-solving skills, creating increasingly sophisticated systems that model a real-world environment with precision and scalability.

Encryption-Decryption Projects

I wanted to learn more about these foundational algorithms for the field of cryptography and get a better understanding on how to securely handle data encryption techniques.

This three simple projects demonstrates the implementation of two classic cryptographic algorithms: the Caesar Cipher and the Vigenère Cipher.

The Caesar-Cipher is one of the simplest and most well-known encryption methods. It works by shifting each letter in the plaintext by a fixed number of positions down or up the alphabet. For example, with a shift of 3, "A" becomes "D," "B" becomes "E," and so on. This project implements the Caesar Cipher to encrypt and decrypt text, showcasing basic encryption principles and the importance of key management.

The program allows users to input plaintext and a shift value, generating encrypted text. It also includes a decryption feature to reverse the process, highlighting the simplicity and limitations of this method.

The Vigenere-Cipher is a more advanced encryption technique that uses a keyword to shift letters in the plaintext. Unlike the Caesar Cipher, which uses a single shift value, the Vigenère Cipher applies different shifts based on the letters of the keyword, making it significantly more secure. This project implements the Vigenère Cipher to demonstrate how encryption can be strengthened by introducing variability in the encryption process.

The program extends the encryption process by using a keyword, providing a more secure way to encrypt and decrypt messages. This demonstrates the practical application of cryptographic techniques in protecting sensitive information.

Together, these implementations illustrate my understanding of encryption algorithms, secure data handling, and the importance of key management in cryptography.

Minesweeper Game Project

A classic Minesweeper game implementation, highlighting problem-solving and game logic development.

The Minesweeper Game Project is a recreation of the classic puzzle game, where the goal is to clear a grid without detonating any hidden mines. I developed this project to strengthen my problem-solving skills and explore game logic development.

In this game, players click on cells to reveal what's underneath. If a cell contains a mine, the game ends. If it's safe, the cell displays a number indicating how many mines are in the surrounding cells. Using these clues, players can strategically clear the grid.

Through this project, I learned how to manage game states, implement grid-based logic, and create an interactive user experience. It was a fun and challenging way to apply programming concepts to a real-world game.
